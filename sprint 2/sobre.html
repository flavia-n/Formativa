Nosso projeto

Somos uma empresa de conteúdo na qual mostramos informações sobre a pandemia do coronavírus, casos, mortes e suspeitos em determinados lugares do mundo.

Nosso objetivo

Nosso objetivo é mostrar o quanto a doença foi prejudicial para o mundo

  • Usamos o Visual Studio Code para a programação do site e o javascript para a programação dos gráficos com os dados
  • Dados obtidos por https://covid19-brazil-api.now.sh/api/report/v1 https://covid19-brazil-api.now.sh/api/report/v1/countries e https://covid19-brazil-api-docs.vercel.app/.
